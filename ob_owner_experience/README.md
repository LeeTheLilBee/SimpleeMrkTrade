# OB Owner Experience Simplification Foundation

This package locks the Observatory owner-experience simplification doctrine.

Each protected room must answer one primary question at first glance.
The default page should show one dominant summary, one principal
recommendation, a few critical indicators, one next action, and
Soulaana's plain-language interpretation.

Details belong behind drawers, tabs, expanders, deep-dive rooms, or
show-details controls. Blanket owner visibility is for viewing only.
Dangerous actions remain separately gated.
